background

Starting from scratch

I began with no coding background — just curiosity and a willingness to build things badly before building them well. Instead of only reading tutorials, I set a rule for myself: every concept I learn has to end up in a real, live project.

That rule turned into this portfolio, which doubles as my sandbox for practicing on-page and technical SEO on an actual hosted site.

how i work
